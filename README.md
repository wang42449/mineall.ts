#### 简介
渐进式,区块组件式,可视化搭建平台,low-code,为个人和企业赋能.可以自由安装搭配扩展/区块，可以用代码进行更复杂开发,可以导入本仓库区块组件用于其它项目，还可以用可视化工具轻松搭建应用程序，致力做自由灵活的产品。

####  目标
让每个用户都能轻松创造,乐于创造.

#### 理念
软件只是一个模板,应该允许用户去自由发挥,改造软件.
